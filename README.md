Spring Boot Crud Application with Thymeleaf, JPA, Spring Security

- Spring MVC
- Spring Boot
- Spring Data JPA / Hibernate
- Spring Security
- Lombok
- Thymeleaf
- Maven
- Twilio
- Liquibase
- MySQL
- Travis
